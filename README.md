## Demo

### 景點搜尋標記
點擊地圖任一點，或使用搜尋功能，查詢景點名稱並標記

### 最短路徑計算
新增景點後可規劃行程，並計算最短路徑，景點可新增刪除

### 路線分析
提供路線資訊，爬升下降高度、里程、行走時間，海拔剖面圖可直接對應地圖景點

### 儲存與匯出GPX和照片
可儲存或匯出GPX和照片，也可公開分享

### 一鍵複製行程，重新規劃專屬路徑
可複製他人規劃好的路線，並重新規劃
